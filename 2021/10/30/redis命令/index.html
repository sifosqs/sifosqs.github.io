redis命令

redis的简单命令

1. Redis：

  1. 简介：Redis是一种数据库。能够存储数据。管理数据的一种软件。
  2. Redis的数据大部分时间是存储在内存的，适合存储频繁访问，数据量比较小的数据。

2.Redis的特点

  1. 支持数据持久化
  2. 支持多种数据结构
    • 不仅支持kay-value类型的数据，同时 还提供list，set，zset，hash
  3. 支持数据备份
    • 支持master-slave模式的数据备份

3.Windows安装Redis

  1. 下载安装包

    https://github.com/dmajkic/redis/releases

  2. 解压到环境文件夹中

  3. 开启Redis

  4. 使用Redis客户端来连接Redis

    记住一句话，Windows下使用确实简单，但是Redis推荐使用Linux去开发

4.Linux安装

  1. 将安装包传到Linux上

  2. 解压Redis到/opt目录

    tar -zxvf redis-5.0.12.tar.gz -C /opt

  3. 编译Redis，进入解压目录，并且执行make命令：

    会报错 ，没有cc命令

    则需要安装gcc

    就需要使用yum进行安装，执行命令；yum -y install gcc

    • 执行 gcc - v查看Linux的内核版本

    再次回到Redis目录执行make命令

    还会报错，因为第一次编译的文件有些会存在，第二次编译就会报错，所以要进行清理工作

    make distclean

  4. 然后执行make install 来安装Redis

  5. 后台启动Redis：usr/local/bin

    redis-server

  6. 启动Redis服务：

    1. 前台启动：在任何目录下执行 redis-server
    2. 后台启动：在任何目录下执行 redis-server &(没改配置文件)
    3. 启动Redis服务时：指定配置文件： redis-server redis.conf &(改配置文件)

  7. 关闭Redis服务：

    1. 通过kill来关闭（暴力不推荐,容易丢失数据）

      ps -ef|grep redis查看pid

      kill -9 pid

    2. 通过Redis命令关闭

      redis-cli shutdown

  8. redis的客户端：通过网络连接到redis服务器，从而实现redis服务器的交互，向服务端发送命令，并且显示redis服务处理结果。

    redis-cli:是redis自带客户端，使用命令redis-cli就可以启动redis的客户端程序。

    redis-cli:默认连接本机（127.0.0.1）上的6379端口上的redis服务

    redis-cli -p 端口号：连接127.0.0.1（本机）指定端口上的服务

    redis-cli -h ip地址 -p 端口号：连接指定主机上的指定端口的redis服务

  9. 退出客户端：在客户端执行命令：exit或者quit

5.redis的基本知识：

  1. 测试redis服务的性能：src里在任何目录下redis-benchmark （了解）

  2. 查看redis服务是否正常运行：

    ping 如果正常——pong

  3. 查看redi服务器s的统计信息：info

    info 查看redis所有的统计信息

    info [信息段]：查看redis服务器 指定的统计信息，如：info replication：查看集群

  4. redis默认使用16个库（==理解==）

    redis中的数据库实例只能有redis服务来创建和维护，开发人员不能修改和自行创建数据库实例；默认情况下，redis会自动创建16个数据库实例，并且给16个数据库进行编号，从0开始到15，使用时，通过编号来使用数据库；可以通过配置文件来制定redis自动创建数据库的个数；每一个数据库实例，本身占用的存储空间是很少的，所以也不会造成存储及空间的太多浪费

    1. 默认情况下，redis客户端连接的是编号位0的数据库实例；
    2. 使用select index 来切换数据库

  5. 查看数据库中key的数目：dbsize

  6. 查看当前数据库到底有哪些k:**keys ***

  7. 清空数据库实例：flushdb

  8. 清空所有数据库实例：flushall

  9. 查看redis中所有的配置信息：**config get ***

  10. 查看redis中指定的配置信息：config get parameter

  11. 清屏；clear

6.redis的5种数据结构

redis是用来存储数据的，程序处理完的数据要存储到redis数据库中，不同特点的数据要存储在redis中不同类型的数据结构中。

  1. 字符串：
  2. list列表（有序，可重复）
  3. set集合（无序，不可重复）
  4. zset:必须后面跟着一个数，来排序，没有数就不要用这个结构
  5. hash

7.redis常用的操作命令：

string：单key:单value

list：单key：多有序value：contacts：a,c,v,

set:单key:多无序value：city:bj,sh,tj

hash 单key：对象（属性：值）：

student：id:1001,name:张三,age:20(k:v)

zset 单key：多有序value：

city:2000 bj,1500 sh ,1000 tj, 1200 cq(有个参照标准，升序排) 有序指的是排序
